# User Profile App 
simple authentication app with flutter, with 4 pages 
* register
* login
* view profile
* edit profile 
# Main target of this app
the main target  is to make clean code using tdd (without test case)
the packages used in this app :
* bloc
* dio 
* freezed
* auto route
* dartz
* retrofit
* get_it
* injectable
* equatable
* image_picker
* image_cropper
* shared_preferences
